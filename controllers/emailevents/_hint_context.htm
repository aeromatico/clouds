Variables de Contexto

Las variables de contexto son los datos que se pasan a las plantillas de correo.

Formato JSON array de strings:

["order_id", "user", "total", "items"]

Ejemplo de uso en plantillas:

Hola {{ user.name }},

Tu pedido #{{ order_id }} ha sido creado.
Total: ${{ total }}

Gracias por tu compra.

Las variables deben ser pasadas cuando se dispara el evento desde el código.
Esta lista es solo informativa para documentar qué variables están disponibles.

Variables comunes:

  • user - Objeto del usuario (name, email, etc.)
  • order_id, invoice_id, ticket_id - IDs de registros
  • total, amount - Montos
  • date, expiration_date - Fechas
